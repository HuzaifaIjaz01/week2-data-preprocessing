# Week 2 - Data Preprocessing and Cleaning Pipeline

## Project Overview
This project focuses on cleaning and preparing raw data for machine learning using Python.

The dataset contains teen mental health information. The project demonstrates how to:
- Handle missing values
- Remove duplicate records
- Encode categorical data
- Apply feature scaling
- Save cleaned datasets

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

---

## Dataset
Teen_Mental_Health_Dataset.csv

---

## Features Implemented
- Missing value handling
- Duplicate removal
- Label Encoding
- Feature Scaling
- Clean dataset export

---

## Output
The cleaned dataset is saved as:

```bash
cleaned_dataset.csv
```
